# About 
包含前後端的甜甜圈販售網站

## 後端部分
使用MySQL資料庫儲存顧客資料、訂單、商品庫存等
## 前端部分
- 帳號系統（註冊、顧客登入、員工登入）
- 訂單系統（用戶僅可瀏覽自己的訂單紀錄）
- 支持訪客瀏覽，但必須登入才能下單(使用js提示訊息提醒用戶登入)
